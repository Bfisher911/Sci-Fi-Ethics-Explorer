# **App Name**: Sci-Fi Ethics Explorer

## Core Features:

- Home Page: Display a welcoming landing page with a 'Dilemma of the Day'.
- Interactive Stories: Browse and filter ethical dilemmas presented as interactive stories.
- Story Navigation: Navigate interactive story segments and display the poll.
- Scenario Analysis: Submit scenarios and receive AI-driven analysis of ethical implications; the AI will be a tool that applies ethical theories.
- AI Chat: Chat with an AI counselor for guidance on ethical dilemmas.
- Authentication: Secure user accounts using Firebase Authentication.
- Community Stories: Display stories submitted by the user community

## Style Guidelines:

- Background Color (--background): HSL: 240 61% 27% Dark Navy Blue (approximately #191970).
- Primary Color (--primary): HSL: 181 100% 74% Electric Blue (approximately #7DF9FF).
- Accent Color (--accent): HSL: 300 100% 50% Vibrant Magenta (#FF00FF).
- Foreground Color (--foreground): HSL: 200 20% 95% A very light, slightly desaturated blue/gray.
- --card: A slightly lighter navy (240 55% 32%) for card backgrounds.
- --border: A distinct blue (240 50% 40%) for borders.
- --ring: Matches the primary Electric Blue, used for focus indicators.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, machined, objective, neutral look.
- Use a set of consistent icons that align with science fiction themes.
- Maintain a clean, organized layout, ensuring all components are easy to access.
- Employ subtle animations when loading data.
